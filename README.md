# flutter_movie_app

TMDB API를 가져와 영화 예약 App입니다.
이 앱에서는 현재 상영중(Now Plaing)인 리스트만 불러왔습니다.
테블릿 기반으로 만들었기 때문에 AVD 또는 일반 핸드폰에서는 UI가 밀리거나 깨질수 있습니다.
